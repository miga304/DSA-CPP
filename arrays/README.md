## Arrays

This folder contains solutions to core array problems.

The focus is on:
- in-place modifications
- optimal time and space complexity
- clear and reliable patterns (two pointers, prefix/suffix, greedy)

### Problems

- Merge Sorted Array  
- Remove Element  
- Remove Duplicates from Sorted Array  
- Remove Duplicates from Sorted Array II  
- Rotate Array  
- Majority Element  
- Product of Array Except Self
- Best Time to Buy and Sell Stock
- Best Time to Buy and Sell Stock II
- Jump Game
- Jump Game II 
- H-Index
- Insert Delete GetRandom O(1)
- Gas Station
- Candy
- Trapping Rainwater

Each solution is implemented in C++ and includes brief comments
explaining the approach and key idea.
